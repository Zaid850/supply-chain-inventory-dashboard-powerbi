# Supply Chain & Inventory Performance Dashboard (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard designed to analyze supply chain
and inventory performance. The dashboard focuses on delivery efficiency, shipping delays,
and profitability to support data-driven business decisions.

The dashboard provides a high-level overview as well as detailed insights into shipping
modes, regional delivery performance, and product-level profitability.

---


## Key Objectives
The primary objectives of this dashboard are to:
- Analyze on-time vs late delivery performance
- Identify shipping modes contributing to delivery delays
- Evaluate product and category-level profitability
- Compare profit performance across delivery statuses

## Key Insights
- Approximately **57% of orders are delivered late**
- **Second Class shipping** has the highest average shipping delay
- Late deliveries still generate **comparable profit per order**
- Fishing and Cleats are among the **top-performing product categories**


---

## Features
- Interactive KPI cards for orders, sales, profit, and average shipping delay
- Delivery performance analysis by shipping mode and region
- Product and category-level profitability analysis
- Dynamic filtering and drill-down capabilities
